# Remidial code-competence

## Cara menjalankan app dengan docker compose

- `docker-compose up -d`

note: pastikan menambahkan file .env di root directory

## Local Endpoints

- Swagger UI
  - `http://localhost`
- API
  - `http://localhost:8000`

- Swagger UI
